GPUPathtracer
=============

physically based path tracer on gpu

![whiteroom](result/whiteroom.png)

特点
----

- 路径追踪器
- 材质(lambertian mirror dielectric roughdielectric roughconduct substrate)
- 面积光源
- 多重重要性采样
- 景深
- 简单的场景加载

依赖库
----
需要CUDA9.1以上

关于场景
------
鉴于场景太大，所以只放了一个简单的测试场景供使用，如果需要，可以联系我将其余场景提供给你
